###Modos de Desenho (Teclas de Ativação)

Ativar um modo de desenho **desativa todos os outros modos de desenho e o modo de seleção**.

| Tecla |        Função            |                         Modo de Desenho                                              |
| **S** | **Alterna Modo Seleção** | Permite interagir com objetos já desenhados (mover, girar, escalar).                 |
| **P** | **Desenho de Polígono**  | Clique esquerdo para adicionar vértices. Pressione **P** novamente para **finalizar**|
| **C** | **Desenho de Círculo**   | **1º clique:** Centro. **2º clique:** Raio (definido pela distância do mouse).       |
| **T** | **Desenho de Triângulo** | **1º clique:** Ponto inicial. **2º clique:** Define o tamanho da base.               |
| **R** | **Desenho de Retângulo** | **1º clique:** Canto inicial. **2º clique:** Canto oposto.                           |

-----------------------------------------------------------------------------------------------------------------------------

Para usar estes comandos, o **Modo Seleção (S)** deve estar ativo e um ou mais objetos devem estar selecionados na tela.

### Movimentação (Teclas Especiais)

Controla o movimento dos objetos selecionados na viewport 2D.

|     Tecla         |               Função                         |
| **Seta Esquerda** | Move para a **esquerda** (`-1.0` no eixo X). |
| **Seta Direita**  | Move para a **direita** (`+1.0` no eixo X).  |
| **Seta Cima**     | Move para **cima** (`+1.0` no eixo Y).       |
| **Seta Baixo**    | Move para **baixo** (`-1.0` no eixo Y).      |

#### Rotação

| Tecla |                                 Função                                          |
| **Z** | Rotaciona o(s) objeto(s) selecionado(s) em **+5 graus** (sentido anti-horário). |
| **X** | Rotaciona o(s) objeto(s) selecionado(s) em **-5 graus** (sentido horário).      |

#### Escala

| Tecla |                         Função                                    |
| **+** (Mais) | Aumenta a escala do(s) objeto(s) selecionado(s) (**10%**). |
| **-** (Menos)| Diminui a escala do(s) objeto(s) selecionado(s) (**10%**). |
